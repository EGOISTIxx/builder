# 🏗️ Минималистичный редактор планов помещений

> Вдохновлён Figma и CAD-инструментами. Позволяет рисовать стены в режиме “line tool” с привязкой (snap) к существующим точкам.

## 📖 Описание

Лёгкий веб-редактор для быстрого создания архитектурных планов и схем помещений. Минималистичный интерфейс, точное управление геометрией и интеллектуальная привязка к узлам делают инструмент удобным как для черновиков, так и для детальной планировки.

## ✨ Возможности

- 📏 **Line Tool**: Рисование стен с контролем длины, углов и толщины
- 🧲 **Smart Snap**: Автоматическая привязка к концам линий, перекрёсткам и направляющим
- 🎨 **CAD-подобный UX**: Горячие клавиши, зум, панорамирование и слои в стиле Figma
- 💾 **Импорт/Экспорт**: Сохранение планов в JSON, экспорт в SVG/PNG
- 🔌 **Программный доступ**: REST API для управления объектами на холсте

## 🚀 Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/EGOISTIxx/builder.git
cd builder

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```
